Solitario

Il solitario che ho creato è il classico Klondike, che ho implementato usando le quaranta carte da briscola e le cinquantasei del Tarocco Piemontese. Le regole del gioco le potete trovare su wikipedia. Credo che questo sia stato il mio primo gioco di carte che abbia implementato in C++ nel lontano 2001, mentre nel 2025 l'ho aggiornato aggiungendo anche la versione Web in Wasm/Escripten.

Il Solitario è disponibile in diverse versioni.

Versione Web

La versione Web (nessuna installazione necessaria) la trovate a questo link.

Versione Windows

La versione Windows la potete scaricare da questo link: Versione Windows

Versione Android

La versione Android (file apk) la potete scaricare da questo link: Versione Android

Codici sorgenti

I sorgenti del programma per tutte le versioni li potete trovare su GitHub

Il progetto utilizza gli stessi sorgenti per i vari target (Windows, Escripten, WSL e Android). La base di tutte le versioni è la libreria C libSdl versione 3, mentre i miei sorgenti sono in C++.
